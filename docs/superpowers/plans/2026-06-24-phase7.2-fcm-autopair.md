## Notes

- **Register local obligatoire** : `npx @liamcottle/rustplus.js fcm-register` (Chrome + Steam),
  puis l'admin colle `androidId`/`securityToken` via `/fcm connect` ou `POST /fcm`.
- Serveur auto-pairé : `guild_id` connu → `addServer` (adoption + défaut) ; sinon orpheline.
  Pas de `channel_id` tant que `/setup` n'est pas passé → `/pop`/`/time`/`!` marchent quand même.
- Idempotence = pas besoin de persister les `persistentId`.
