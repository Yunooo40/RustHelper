# Phase 8.3 — Heli & Bradley détruits (markers Explosion) — Plan

Date : 2026-06-24 · Branche : `claude/hopeful-hypatia-uyyjwp` · Suite de la Phase 8.2.

Objectif : étendre le poller de markers pour annoncer **Heli abattu** et **Bradley détruit**
(les deux droppent des caisses), toujours **sans plugin**, via les marqueurs **Explosion** de
`getMapMarkers`. Réutilise les clés d'event existantes `helicopter` / `bradley` et le même
pipeline d'ingestion (`recordRustEvent`).

## Idée centrale
Un marker **Explosion** (AppMarker.Type 2) = un Patrol Heli OU un Bradley qui vient de tomber.
Le marker seul ne dit pas lequel → on **corrèle dans le même poll** :
- explosion qui apparaît **+ un heli qui disparaît** le même poll → **heli détruit**
  (et on **supprime le « left »** en double de cet heli) ;
- explosion **sans** heli disparu → **Bradley détruit**.

Un heli qui quitte la carte disparaît **sans** explosion → reste un simple `helicopter left`.
